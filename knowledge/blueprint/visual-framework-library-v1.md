# NARRATIIVE VISUAL FRAMEWORK LIBRARY v1

## STATUS

Active

## VERSION

1.0

## OWNER

Narratiive

## PURPOSE

This library defines the fixed visual frameworks used to communicate strategy in the Narratiive Growth Blueprint.

It is the design and communication reference for blueprint population.

## FIXED DESIGN SYSTEM

Use a premium strategic visual language.

The work should be:

- clear
- founder-grade
- commercial
- evidence-led
- visually disciplined

## FRAMEWORK CATALOGUE

### 1. Ecosystem Map of Forces

Show the market forces acting on the client.

### 2. Growth Friction Framework

Show where growth is being blocked.

### 3. Risk Matrix

Show the main strategic risks and their severity.

### 4. Opportunity Landscape

Show the full set of opportunities available.

### 5. Opportunity Prioritisation Model

Rank opportunities by leverage.

### 6. Audience Ecosystem

Map the audience and its relationships.

### 7. Audience Gap Model

Show where the client is missing demand.

### 8. Positioning Map

Show where the brand sits relative to competitors.

### 9. We Are / We Are Not Framework

Clarify the strategic boundaries of the brand.

### 10. Tension Framework

Show the tensions the narrative must resolve.

### 11. Narrative Framework

Define the story engine.

### 12. Messaging Pyramid

Structure the message from core idea to supporting proof.

### 13. Asset Ecosystem

Show the asset system that supports the campaign.

### 14. Channel Role Map

Define the role of each channel.

### 15. Attention Framework

Explain how attention is earned and sustained.

### 16. Campaign Factory

Show how the campaign becomes repeatable execution.

### 17. Roadmap

Show the strategic path forward.

### 18. Measurement Pyramid

Define the measurement stack from brand to performance.

### 19. Single Statement

Capture the one strategic statement the client should remember.

## SUPPORTING OBJECTS

### Title Lockup

Use a clean title lockup when the framework needs a formal header.

### Contact Block

Use a restrained contact block for owner and attribution details.

## LAYOUT TYPE KEY

- A: Full-bleed statement
- B: Split visual and text
- C: Multi-panel comparison
- D: Matrix or grid
- E: Sequential framework

## SELECTION LOGIC

Choose the framework that best answers the strategic question being asked.

- Market forces question -> Ecosystem Map of Forces
- Growth blockers question -> Growth Friction Framework
- Risk question -> Risk Matrix
- Opportunity question -> Opportunity Landscape
- Priority question -> Opportunity Prioritisation Model
- Audience question -> Audience Ecosystem or Audience Gap Model
- Positioning question -> Positioning Map
- Boundaries question -> We Are / We Are Not Framework
- Tension question -> Tension Framework
- Story question -> Narrative Framework
- Messaging question -> Messaging Pyramid
- Asset question -> Asset Ecosystem
- Channel question -> Channel Role Map
- Attention question -> Attention Framework
- System question -> Campaign Factory
- Roadmap question -> Roadmap
- Measurement question -> Measurement Pyramid
- Summary question -> Single Statement
