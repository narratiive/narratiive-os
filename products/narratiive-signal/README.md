# Narratiive Signal

The Narratiive Signal is the canonical outreach product for Narratiive.

It is an unsolicited strategic gift: a concise, evidence-led point of view on a business, paired with a distinctive creative possibility that demonstrates how Narratiive thinks before a commercial relationship begins.

## Canon

Read these files in order before generating or reviewing any Signal:

1. `01-product-spec.md` — what the product is and what it must contain
2. `02-editorial-style-guide.md` — how it must sound
3. `03-generation-contract.md` — the execution contract for Claude or any future model
4. `04-quality-rubric.md` — the scoring standard
5. `05-review-checklist.md` — Tony's approval gate

## Naming

- Internal workflow name: `Opportunity Card Pipeline`
- External product name: `Narratiive Signal`

## Operating principle

Tony orchestrates. Claude researches, reasons and writes. Higgsfield renders speculative creative. Tony stores, validates, versions and routes the finished work for human approval.

No agent should rely on memory or an embedded mega-prompt when creating a Signal. The files in this folder are the source of truth.
